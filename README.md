# vagrant-sandbox

Sandbox for test some tools by vagrant

## Getting start

First, choose OS.

```bash
.
├── centos-8-2011.0
├── centos-8-latest-cockpit
├── centos-8-latest-gui
├── centos-8-latest-slim
├── debian-contrib-buster64-10.20211018.1
├── kalilinux-v2021.2.0
├── ubuntu-focal64-latest
└── ubuntu-focal64-latest-gui
```

Second, Vagrant up.

```bash
vagrant up
```
